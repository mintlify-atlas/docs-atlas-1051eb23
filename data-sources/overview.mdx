### Distributed SQL & Analytics

- **CockroachDB** - Distributed SQL database
- **Presto/Trino** - Distributed SQL query engine
- **Apache Drill** - Schema-free SQL query engine
- **Apache Ignite** - In-memory SQL database
- **Apache Kylin** - OLAP engine for big data
- **ClickHouse** - Column-oriented database for analytics
- **DuckDB** - In-process SQL OLAP database
- **Exasol** - High-performance analytics database
- **Impala** - SQL query engine for Hadoop
- **Apache Hive** - Data warehouse infrastructure
- **Apache Phoenix** - SQL layer over HBase
- **Pinot** - Real-time distributed OLAP datastore
- **RisingWave** - Distributed SQL streaming database
- **Vertica** - Columnar analytics platform
- **MemSQL/SingleStore** - Distributed SQL database
