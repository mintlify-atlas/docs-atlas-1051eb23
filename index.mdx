      <p className="max-w-2xl text-lg text-gray-600 dark:text-gray-400">
        Follow these steps to set up Redash and create your first dashboard
      </p>
    </div>

    <Steps>
      <Step title="Deploy Redash">
        Use Docker Compose to get Redash running quickly:

        ```bash
        curl -O https://raw.githubusercontent.com/getredash/setup/master/setup.sh
        chmod +x setup.sh
        ./setup.sh
        ```

        Access Redash at `http://localhost:5000` and complete the setup wizard.
      </Step>

      <Step title="Connect Your Data Source">
        Navigate to **Settings** → **Data Sources** and click **New Data Source**. Choose from 35+ supported sources including:

        - PostgreSQL, MySQL, SQL Server
        - BigQuery, Redshift, Snowflake
        - MongoDB, Elasticsearch, DynamoDB
        - Prometheus, Google Analytics, and more

        Enter your connection details and test the connection.
      </Step>

      <Step title="Write Your First Query">
        Click **Create** → **Query** and select your data source. Use the schema browser to explore tables and write SQL:

        ```sql
        SELECT
          date_trunc('day', created_at) as day,
          count(*) as signups
        FROM users
        WHERE created_at >= current_date - interval '30 days'
        GROUP BY 1
        ORDER BY 1
        ```

        Execute the query and see results instantly.
      </Step>

      <Step title="Create Visualizations">
        Click **+ New Visualization** and choose from multiple chart types:

        - Line, Bar, Area charts
        - Pie, Scatter, Bubble charts
        - Pivot tables, Counters, Maps

        Customize colors, labels, and formatting. Add visualizations to a dashboard for a complete view of your data.
      </Step>
    </Steps>
